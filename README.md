# 마피아42 카드 뽑기 시뮬레이터

마피아42 게임의 카드 뽑기 시뮬레이터입니다.

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다.
2. 터미널 입력창에 명령어를 입력합니다:
   - `N TIER`: N티어 카드 1개를 뽑습니다 (예: `5 TIER`)
   - `N M TIER`: M티어 카드 N개를 뽑습니다 (예: `3 5 TIER` = 5티어 카드 3개)

## 기능

- 1티어 능력: 악행(41.7%) 또는 정의(58.3%)
- 2티어 능력: 직업별 확률에 따른 직업 선택
- 3티어 능력: 직업에 따라 다른 능력 풀 (마피아/짐승인간은 광기 포함)
- 4-6티어 능력: 중복 없이 선택되는 능력 (직업별 고유 능력 포함)

## 티어별 설명

- **1티어**: 악행 또는 정의
- **2티어**: 직업 (마피아, 경찰, 의사 등)
- **3티어**: 게임 외적 버프 능력
- **4-6티어**: 게임 내 능력 (중복 없음)
